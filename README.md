# PianoGame (Piano Tiles Clone)

ブラウザ上でプレイ可能な「ピアノタイル」風の音楽ゲームです。
シングルプレイで自己ベストを目指せるほか、オンラインでのリアルタイム対戦機能、モード別のランキング機能、そしてきめ細かいカスタマイズ機能を備えています。

## 機能・特徴

- **5つのゲームモード**:
  - Normal (4鍵)
  - Hyper (6鍵)
  - Another (8鍵)
  - Leggendaria (9鍵)
  - DP (16鍵)
- **オンライン対戦**:
  - 同じモードを選んだプレイヤー同士でリアルタイムにマッチングします。
  - 相手のプレイスクリーンが横にリアルタイムで表示され、臨場感のある対戦が楽しめます。
- **ランキング機能**:
  - プレイ終了時に自己ベストが更新された場合、サーバーに送信され記録されます。
  - データは `rankings.json` に保存され、タイトル画面からいつでもモード別に閲覧可能です。
- **詳細なカスタマイズ (ローカルストレージに保存)**:
  - **キー設定**: 各列（タイル）に対して好きなキーボードのキーを割り当て可能です。
  - **カラー設定**: 各列のタイルの色を自由に設定できます。
  - **タイルの長さ**: 降ってくるタイルの縦の長さを変更できます。
  - **プレイ画面の横幅**: ウィンドウサイズや好みに合わせて、ゲーム画面の横幅を調整できます。
  - **判定ラインの位置**: 判定ライン（ノーツを叩く基準線）の上下位置を調整し、表示の有無も切り替え可能です。
  - ライブプレビュー画面を見ながら、自分に合った最適な環境を構築できます。

## 動作要件

- Node.js (v14以上推奨)
- npm
- Webサーバー (Apache, Nginx など ※Node.js単体でのホスティングも可能です)

## デプロイ・起動方法

### 1. リポジトリの配置とパッケージのインストール
プロジェクトのディレクトリで以下のコマンドを実行し、必要なパッケージ（`express`, `socket.io` 等）をインストールします。

```bash
cd /path/to/PianoGame
npm install
```

### 2. バックエンドサーバー（Socket.io）の起動
マルチプレイの通信およびランキング保存のために、Node.jsサーバーを立ち上げる必要があります。
以下のコマンドを実行するか、`pm2`や`systemd`などのプロセス管理ツールを使用してバックグラウンドで起動させてください。

```bash
npm start
# または node server.js
```
> ※デフォルトでは `0.0.0.0:3040` で起動します（`server.js` 参照）。
> ※ランキングデータは `rankings.json` に保存されます。Node.jsのプロセスがこのファイルに対して書き込み権限（例: `chmod 666 rankings.json`）を持っていることを確認してください。

### 3. フロントエンドの配信 (Webサーバーの設定)

フロントエンドのファイル群（`index.html`, `game.js`, `style.css`）をブラウザで表示できるようにします。

**パターンA: Apache や Nginx を使用する場合（推奨）**
現在のディレクトリ（`/var/www/html/PianoGame` など）にアクセスできるようにWebサーバーを設定してください。
ゲーム画面にアクセスすると、クライアント側のJSが自動的に同じホストのポート`3040`（Node.jsサーバー）へSocket接続を行います。

**パターンB: Node.js だけで配信する場合**
`server.js` には `app.use(express.static(path.join(__dirname, 'public')));` の記述があるため、プロジェクト内に `public` フォルダを作成し、フロントエンドファイル（`index.html`, `game.js`, `style.css`）を移動させれば、ポート`3040`だけで静的ファイルとWebSocketの両方を配信することができます。
ブラウザからは `http://ホスト名:3040` でアクセスできるようになります。

## 遊び方

1. ブラウザでゲームページにアクセスします。
2. 名前を入力し、遊びたいモード（Normal / Hyper / Another / Leggendaria / DP）を選択します。
3. 初めてプレイする場合や操作感を変えたい場合は、**「キー・カラー設定」** ボタンを押して、各タイルの色・キー配置・長さ・判定ラインなどをカスタマイズします。
4. **「シングルプレイ」**で一人で遊ぶか、**「オンライン対戦」**で対戦相手をマッチングします。
5. 上から降ってくるタイルが一番下のラインに到達するタイミングに合わせて対応するキーを押します。ラインを越えてしまうとゲームオーバーです！
