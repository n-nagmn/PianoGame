# PianoGame (Piano Tiles Clone)

ブラウザで遊べる「ピアノタイル」風の音楽ゲームです。シングルプレイに加え、リアルタイムのオンライン対戦やランキング機能、キー＆カラーのカスタマイズ機能を備えています。

## 機能・特徴

- **3つのゲームモード**:
  - Normal (4鍵)
  - Hyper (6鍵)
  - Another (8鍵)
- **オンライン対戦**:
  - 同じモードを選んだプレイヤー同士でリアルタイム対戦が可能。
  - 相手のプレイスクリーンがリアルタイムに表示されます。
- **ランキング機能**:
  - プレイ終了時にスコアがサーバーに送信され（SQLiteに保存）、モード別にランキングを表示。
- **キー・カラーカスタマイズ**:
  - 各列（タイル）に対して好きなキーと色を割り当て可能。設定はブラウザ（ローカルストレージ）に自動保存されます。

## 動作要件

- Node.js (v14以上推奨)
- npm
- Webサーバー (Apache, Nginx など ※またはNode.js単体でホスティングも可)

## デプロイ・起動方法

### 1. リポジトリの配置とパッケージのインストール
プロジェクトのディレクトリで以下のコマンドを実行し、必要なパッケージ（`express`, `socket.io`, `sqlite3`）をインストールします。

```bash
cd /path/to/PianoGame
npm install
```

### 2. バックエンドサーバー（Socket.io）の起動
マルチプレイの通信およびランキングデータベースのために、Node.jsサーバーを立ち上げる必要があります。
以下のコマンドを実行するか、`pm2`や`systemd`などのプロセス管理ツールを使用してバックグラウンドで起動させてください。

```bash
npm start
# または node server.js
```
> ※デフォルトでは `0.0.0.0:3040` で起動します（`server.js` 参照）。

### 3. フロントエンドの配信 (Webサーバーの設定)

フロントエンドファイル（`index.html`, `game.js`, `style.css`）をブラウザで表示できるようにします。

**パターンA: Apache や Nginx を使用する場合（推奨）**
現在のディレクトリ（`/var/www/html/PianoGame` など）にアクセスできるようにWebサーバーを設定してください。
ゲーム画面にアクセスすると、クライアント側のJSが自動的に同じドメインのポート`3040`（Node.jsサーバー）へSocket接続を行います。

**パターンB: Node.js だけで配信する場合**
`server.js` には `app.use(express.static(path.join(__dirname, 'public')));` の記述があるため、プロジェクト内に `public` フォルダを作成し、その中に `index.html`, `game.js`, `style.css` を移動させれば、ポート`3040`だけでフロントエンドとバックエンドの両方を配信することができます。
ブラウザからは `http://ホスト名:3040` でアクセスできるようになります。

## 遊び方

1. ブラウザでゲームページにアクセスします。
2. 名前を入力し、遊びたいモード（Normal / Hyper / Another）を選択します。
3. 初めてプレイする場合やキーを変えたい場合は、**「キー・カラー設定」** ボタンを押して、各タイルの色と対応するキーボードのキーをカスタマイズします。
4. **「シングルプレイ」**（またはスペースキー）で一人で遊ぶか、**「オンライン対戦」**で対戦相手をマッチングします。
5. 上から降ってくるタイルに合わせてキーを押し、スコアを稼ぎます。一番下のラインを通過するまでに押せないとゲームオーバーです！
